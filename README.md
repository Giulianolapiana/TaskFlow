# TaskFlow — Gestor de Tareas

TaskFlow es una aplicación web (PWA) de gestión de tareas rápida, eficiente y totalmente independiente centrada en la productividad personal. Construida con una filosofía minimalista en mente: **sin bases de datos, sin backend y sin tiempos de carga.** Todo el almacenamiento y la lógica ocurren directamente en tu navegador.

## 🌐 Demo en Vivo
Puedes acceder y utilizar la aplicación desplegada en: **[https://task-flow-gules-one.vercel.app/](https://task-flow-gules-one.vercel.app/)**

## 🚀 Características Principales
- **Arquitectura "Frontend Only"**: HTML, CSS y JavaScript Vanilla puros agrupados en unos pocos archivos.
- **Soporte Offline Puro (PWA)**: Gracias a la implementación de un *Service Worker*, TaskFlow puede instalarse como aplicación nativa (Windows/Android/iOS) y funciona completamente en Modo Avión.
- **Vistas Especializadas**: Diaria, Semanal y Mensual (con mini calendario interactivo).
- **Métricas y Estadísticas**: Seguimiento de rendimiento visual en tiempo real y categorización de tareas completadas.
- **Interactividad Avanzada**: Mover tareas entre diferentes días con *Drag & Drop*.
- **Alertas Visuales y Push**: Resaltado automático para tareas con fechas de entrega vencidas.
- **Backup Local**: Botones nativos para exportar e importar la base de datos de tareas en formato JSON.
- **Personalización**: Temas Claro / Oscuro integrados y persistentes.

## 🛠️ Tecnologías
- HTML5
- CSS3 (Variables, CSS Grid, Flexbox, Keyframes)
- JavaScript Vanilla (Almacenamiento Local, APIs del DOM, Drag & Drop API)
- Service Workers y Web App Manifest
